In this project, I explored the basics of Denial-of-Service (DoS) attacks by creating a controlled lab environment using virtual machines.

Setup: 
Configured a Windows Virtual Machine as the target system, set up a Kali Linux Virtual Machine as the attacker. ensured both systems were connected within a safe, isolated network

Attack Simulation:Used tools like hping3 to simulate a DoS attack, generated continuous traffic to overwhelm the target machine

Monitoring and Analysis:Captured and analyzed network traffic using Wireshark, Observed how the attack affected system performance and network behavior

Mitigation technique:Applied basic defensive measures to reduce the impact of the attack

Learning Outcomes:Gained hands-on experience with DoS attack simulation, understood how attackers exploit network vulnerabilities, learned the importance of network monitoring and basic mitigation strategies
